# Funções, Exceções e Exercícios

Este repositório contém exemplos de funções, tratamento de exceções e exercícios em Python. Ele foi criado com o objetivo de reunir códigos úteis para estudo, prática e aprofundamento dos conceitos fundamentais da linguagem.

## Conteúdo

- Exemplos de funções personalizadas
- Demonstrações de tratamento de exceções (`try`, `except`, `finally`)
- Exercícios práticos para fixação de conceitos

## Estrutura dos arquivos

Os arquivos estão organizados conforme o tema/atividade abordada. Cada arquivo contém comentários explicativos para facilitar o entendimento do código.

## Como usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/daniellynuness/fun-es-e-exce-es.git
   ```
2. Navegue até o diretório do projeto:
   ```bash
   cd fun-es-e-exce-es
   ```
3. Execute os arquivos Python:
   ```bash
   python nome_do_arquivo.py
   ```

## Requisitos

- Python 3.x instalado ([Download aqui](https://www.python.org/downloads/))
